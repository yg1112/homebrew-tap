# Homebrew Tap for Mint

[Mint](https://mint.dzgapp.com) is a storage intelligence tool for macOS.

## Install

```bash
brew tap yg1112/tap
brew install mint-cli
```

## Usage

```bash
mint scan          # Scan for reclaimable space
mint scan --all    # Full scan (dev + app + junk)
mint status        # Disk trends and activity summary
mint why           # What categories are growing fastest
mint watch         # Live stream of file operations
mint rules list    # List cleanup rules
```
